## 📋 وصف المشروع

نظام تقييم إلكتروني متكامل لعيادات طبية، يقيس أداء العيادة عبر محاور متعددة باستخدام منهجية علمية مبنية على:

- **قيم الإجابات:** {0, 40, 100}
- **أوزان التأثير:** high (×1.5) / medium (×1.0) / low (×0.5)
- **طبقات الأسئلة:** Layer A (يُحتسب) / Layer B (كمين)
- **العقوبات:** ديناميكية 10%–30% (تُفعّل فقط إذا target=100)
- **التصنيف:** Q1/Q2/Q3/Q4 (لا يوجد A/B/C/D/F)
- **KPIs:** TFI / TAP / PRP / PLI
