## 📋 وصف المشروع

هذا المشروع عبارة عن تطبيق ويب ثابت لتقييم أداء العيادات الطبية عبر نموذج إلكتروني تفاعلي. يُستخدم لتقييم الجوانب الأساسية لرحلة المريض وأداء العيادة بناءً على أسئلة منظمة ومصنفة وفق نظام درجات علمي.

### المميزات الأساسية

- تقييمات متعددة: رحلة المريض وأداء العيادة
- منطق تقييم مركزي ومُدار عبر JavaScript
- بيانات التقييم خارجية في ملفات JSON
- واجهة بسيطة مع الحفاظ على التصميم الأصلي
- اختبارات تلقائية لحماية السلوك الأساسي للتطبيق

### المنهجية المستخدمة

- قيم الإجابات: {0, 40, 100}
- أوزان التأثير: high (×1.5) / medium (×1.0) / low (×0.5)
- طبقات الأسئلة: Layer A (مُحتسب) / Layer B (كمين)
- عقوبات ديناميكية: 10%–30% عند تفعيل target=100
- تصنيف النتائج: Q1 / Q2 / Q3 / Q4
- مؤشرات الأداء: TFI / TAP / PRP / PLI

## 📁 هيكل الملفات

- index.html: الصفحة الرئيسية
- patient-journey.html: نموذج تقييم رحلة المريض
- clinic-performance.html: نموذج تقييم أداء العيادة
- assets/js/app.js: منطق التطبيق والتبديل بين المشاهد
- assets/js/engine.js: منطق الحساب والتصنيف
- data/config.json: تعريف الأسئلة والهيكل
- data/report_texts.json: النصوص العربية والملصقات الخاصة بالتقارير
- tests/app-view-toggle.test.js: اختبار تلقائي لحماية التبديل بين المشاهد
